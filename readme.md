# API de Livros
# Objetivo 
- Criar uma API que disponibiliza a consulta, criação, edição e exclusão de livros
# URL base
- localhost
# Endpoints 
   - localhost/livros [GET]
   - localhost/livros [POST]
   - localhost/livros/id [GET]
   - localhost/livro/id [PUT]
   - localhost/livro [DELETE]
# Recursos
- Livros
